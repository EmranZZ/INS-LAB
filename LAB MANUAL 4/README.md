# Lab Manual 4: Programming Symmetric & Asymmetric Crypto

This project implements symmetric and asymmetric cryptographic operations using Python.

## Features

- AES Encryption/Decryption (128 & 256 bit)
- AES Modes: ECB and CFB
- RSA Encryption/Decryption
- RSA Digital Signature & Verification
- SHA-256 Hashing
- Execution Time Measurement
- Key Size vs Time Graph Plotting

---

## Installation

Create virtual environment:
    python3 -m venv venv
    source venv/bin/activate
    pip install pycryptodome matplotlib

---

## Run
    source venv/bin/activate
    python3 main.py

---

## Directory Structure

- keys/ → Stored keys
- inputs/ → Input files
- outputs/ → Encrypted outputs & signatures
- screenshots/ → Program output screenshots
