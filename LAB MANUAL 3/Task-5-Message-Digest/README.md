# Task 5: Generating Message Digest

## Algorithms Used
- MD5
- SHA1
- SHA256

## Commands Used

openssl dgst -md5 message.txt
openssl dgst -sha1 message.txt
openssl dgst -sha256 message.txt

## Observation

MD5 produces 128-bit hash.
SHA1 produces 160-bit hash.
SHA256 produces 256-bit hash.

Longer hash length increases security.
